h3. About Jamoma 0.6

Jamoma 0.6 will be introducing major changes to how Jamoma Modules are structured, introducing true model-view-controller separation. An introduction to Jamoma 0.6 is provided in the following video:

p=. <iframe src="http://player.vimeo.com/video/35026525" width="500" height="281" frameborder="0" webkitAllowFullScreen mozallowfullscreen allowFullScreen></iframe> 

Some details concerning the development of Jamoma 0.6 can be found "here":http://redmine.jamoma.org/projects/modular/wiki/Jamoma_06. Please note that all modules developed using Jamoma 0.5 will need to be updated in order to work with Jamoma 0.6.
